# 🚀 Runar — Gerador de Propostas Inteligentes para Freelancers

**Micro SaaS pessoal** que usa **Inteligência Artificial (Gemini)** para criar, editar, enviar e rastrear propostas comerciais profissionais em minutos.

Plataforma completa que transforma o envio de propostas em um **funil de vendas inteligente**: gera propostas com estrutura otimizada, melhora o texto com IA, cria link público exclusivo e rastreia todas as visualizações do cliente em tempo real (data, hora, dispositivo e IP).

---

### ✨ Principais Funcionalidades

- 🤖 **Geração e edição de propostas** com IA (Gemini) – estrutura pronta + melhoria automática de texto  
- 🔗 **Link público exclusivo** para cada proposta (experiência moderna de compartilhamento)  
- 👁️ **Rastreamento em tempo real** de visualizações (data, hora, dispositivo, IP)  
- 📊 **Dashboard analítico** com métricas de engajamento e follow-up inteligente  
- 📄 **Exportação em PDF** profissional  
- 👥 **Gestão completa de clientes** e histórico de propostas  
- 💳 **Sistema de planos e assinatura** (multi-tenancy)  
- 📱 **Interface 100% responsiva** (HTML + CSS + JS)
